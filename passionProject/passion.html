Web Development

I like Programming because I like to see my ideas for websites in my head turned into a reality. First I get whatever picture or text I plan to put on the site. Second I make a wireframe in something like Figma. Third, I make it in HTML and CSS.

I was introduced to programming by freecodeCamp and Code Academy. FreecodeCamp and Code Academy taught me how to program in HTML CSS and JS. I have since gotten better at HTML and CSS, and designed a few basic sites. The website consisted of two basic online stores and an art gallery. Today I am still learning JS from FreecodeCamp and Codecademy.

I like the amount of money I could make while being a programmer. I also like the freedom, flexibility and ability to be a freelancer while working remote from anywhere with an internet connection.

The growth potential financially, and the freedom and flexibility to work anywhere.

Pros to Web Development

  • Money, According to Glassdoor the average salary front of a end web developer makes $86,403.
  • Flexibility and Freedom, as a developer you will often have flexible time and the freedom to work from any location.
  • Remote work, Being a web developer does note require you to be in a pysical location. Web developer can work from anywhere in the world with an internet connection. Many campanies today allow remote work for developers
  • Growth potential, There are many different paths as a developer, with a lot of financial growth opportunities. For example Freelance, contract, normal job, ect. No matter what path you choose there is still growth potential.
